# Workforce Attrition Analysis

## 📌 Project Overview

This project analyzes employee attrition data to identify key factors influencing employee turnover and highlight high-risk areas within the organization.

---

## 🎯 Objective

To understand patterns of employee attrition and provide insights that help improve employee retention.

---

## 🛠️ Tools Used

* Python
* Pandas
* Seaborn
* Matplotlib

---

## 🔍 Key Insights

* Employees with **0–2 years tenure** have highest attrition
* **Sales Representatives** show highest attrition (~40%)
* Employees working **overtime** have significantly higher attrition
* **Frequent travel** increases attrition risk
* Poor **work-life balance** leads to higher attrition
* Employees with **lower income** tend to leave more

---

## 💡 Conclusion

Attrition is mainly driven by early career stage, workload, and compensation factors. Addressing these areas can help improve employee retention.
